# Mobile Humanization Notes

## Findings

- Mobile needed more human pacing, not a new layout system.
- Evidence cards were readable, but lacked enough context to feel grounded.
- CTA language leaned on free/low-commitment framing instead of consultive readiness.
- Trust signals were present, but scattered between footer, evidence, and hero.

## Changes

- Added mobile-safe trust layer tracking through `Mobile Trust Engagement`.
- Evidence metadata now has short context, note, decision, and redaction lines that wrap inside cards.
- CTA labels now favor "evaluación técnica" and "casos técnicos".
- Footer surfaces engagement models, secure communications, and responsible disclosure without adding fake partner theater.

## Ongoing Checks

- Evidence context rows must wrap cleanly under 390px width.
- Trust CTAs must remain reachable without scrolling past multiple repeated cards.
- Operational notes should stay short enough to scan on mobile.
- No desktop-only artifact should be the only source of trust context.
